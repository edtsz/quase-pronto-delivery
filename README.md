# Quase Pronto 
Gestão de pedidos

## Itens
- pizzas com sabores variados
- batatinha frita

## Preço
- tradicional: R$ 40,00
- especial: R$ 50,00

## Entrega
- centro: R$ 10,00
- bairro: R$ 15,00
- outros: R$ 20,00

## Gestão
Ao finalizar a noite (sistema) ele precisa saber:
- quantas pizzas foram vendidas;
- qual o valor que tem que pagar para o motoboy;
- o faturamento total;
- o faturamento líquido;
